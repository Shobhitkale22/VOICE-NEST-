# VOICE-NEST-
VoiceNest is a secure voice recording application with encrypted local storage, AI transcription, and cloud sharing capabilities.
VoiceNest is a secure voice recording application currently being developed as a Proof of Concept (POC).

The application enables users to securely record voice notes, store them in encrypted form, and play them back within the application. Future versions will include AI-powered transcription, private cloud synchronization, and secure voice sharing.

## Features (Planned)

- 🎤 Voice Recording
- 🔒 Encrypted Local Storage
- ▶️ In-App Playback
- 📝 AI Speech-to-Text
- ☁️ Cloud Synchronization
- 👥 Secure Sharing
- 📱 Cross-Platform Support (React Native)
