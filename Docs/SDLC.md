📄 SOFTWARE DEVELOPMENT LIFE CYCLE (SDLC)
Project Name

Secure Voice Vault – Proof of Concept (POC)

Development Methodology

Agile (Iterative & Incremental Development)

Why Agile?

We chose Agile because:

Requirements may evolve after feedback from the founder.
We want to deliver working features quickly.
We can improve the application after each milestone.
It reduces development risk.
